# Context

## プロジェクト名

ズボラカレンダー

## 目的

予定、課題、バイト、リマインドの登録が面倒な問題を減らす。自然文やリンクを投げるだけで予定候補を作り、確認して登録や通知につなげる。

## MVP

MVP Phase 1 は自然文を予定候補オブジェクトへ変換するパーサ。

対象例:

- 明日17時からバイト
- 今日21時に課題
- 6/28 13:00 物理レポート締切
- 来週月曜 朝にゴミ出し

## 将来構想

- Discord から自然文で予定候補を作る
- Gmail から課題、締切、面談などを拾う
- Google Calendar に確認後登録する
- Notion にタスクや文脈を保存する
- iPhone や Discord で通知する

## 想定ユーザー

予定や課題の登録が面倒で、Discord やメールに流れてきた情報をその場で軽く投げておきたい個人ユーザー。

## 重要な制約

- 初期段階では巨大実装しない。
- 外部 API 連携は明示指示があるまで実装しない。
- API キーやトークンをコードに直書きしない。
- タイムゾーンは `Asia/Tokyo` を前提にする。
- 完璧な日本語自然文解析は目指さない。
